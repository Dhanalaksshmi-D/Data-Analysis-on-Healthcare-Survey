Healthcare Survey Data Analysis

This project is a mini data analysis study based on a healthcare survey conducted using Google Forms. 
The responses were collected, downloaded as a CSV file, and analyzed using Python, Pandas, NumPy, and Matplotlib. 
The goal of this project is to understand healthcare habits, accessibility, satisfaction levels, and overall wellness trends among the respondents.

Technologies & Libraries Used

Python

Pandas – Data cleaning & analysis

NumPy – Numerical operations

Matplotlib / Seaborn – Data visualization

Jupyter Notebook – For running and documenting analysis
